# Mongoose-Scraper

## Description
Use Cheerio to scrape and Mongoose to persist news articles from the https://news.ycombinator.com/ web site,  
this web app then lets users create or delete note on each of the articles. 
  
 ## Tech Stack
 
* express
* express-handlebars
* mongoose
* body-parser
* cheerio
* request
* handlebars
